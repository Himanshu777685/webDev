# 📝 ToDo App

A simple ToDo list web application made with HTML, CSS, and JavaScript.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent with localStorage (optional)

---

## 📂 Files

- `index.html`
- `style.css`
- `script.js`
  
---

## 🛠️ How to Run

1. Clone the repo
2. Open `index.html` in your browser
